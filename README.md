# Offline_speech_synthesis

## version1.1
> 更新线程方式, 只用一个线程合成音频, 节省资源, 提高灵活性.  
> 添加reset, 快速重置, 重新播放音频.  
> 优化合成速度.  
> 解决多音字问题.  
> 添加频率设置, 可以变换频率, 改变声音, 值越小, 声音越浑厚, 值越大, 声音越尖锐, 10000-22050.  
> 暂时不支持小数, 因为部分场景考虑小数比较复杂.  
> 适配python2.7.  

## version2.0
> 优化合成性能
